🚀 Overview
This project demonstrates how to create a responsive, text-based chatbot. By leveraging dictionary-based lookups and string processing, the bot maintains an engaging conversation flow, handles basic emotional input, and manages session state.

🛠 Features
Dynamic Response System: Uses a dictionary-based structure for efficient and scalable keyword matching.

Conversational Flow: Includes polite, follow-up questions to encourage continued interaction.

Input Normalization: Automatically handles case sensitivity and extra whitespace for a smoother user experience.

Fallback Logic: Provides a default response for unrecognized input to keep the conversation going.

💻 Tech Stack
Language: Python

Key Concepts:

Functions for modularity

Dictionaries for response mapping

while loops for continuous session management

String manipulation (.lower(), .strip())

Task 2 - CodeAlpha_HangMan Game
Hangman Game - Python Programming
This project is a text-based implementation of the classic Hangman game, developed as part of my Python Programming Virtual Internship at CodeAlpha.

🚀 Overview
The objective of this project was to build a functional, command-line version of the Hangman game. It reinforces core programming concepts such as loops, conditional logic, and string manipulation while adhering to a specific set of game constraints.

🛠 Features
Dynamic Word Selection: Randomly selects a word from a predefined list.

Input Validation: Ensures the user inputs only valid, single letters and prevents duplicate guesses.

Progress Tracking: Displays the current state of the word (revealed letters vs. underscores).

Game Constraints: Limits the player to 6 incorrect guesses before the game concludes.

Clean CLI Interface: Simple and intuitive console-based interaction.

💻 Tech Stack
Language: Python

Concepts Applied:

random module for word selection

while loops for game state management

if-else blocks for win/loss logic and validation

String formatting and lists for data handling
